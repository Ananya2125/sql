# College Management Database

This project implements a simple College Management Database using MySQL, including tables for students, courses, faculty, and enrollments. The project includes SQL scripts for creating the database and inserting sample data.

## Features
- Students Table
- Courses Table
- Faculty Table
- Enrollments Table (Mapping Students to Courses)
